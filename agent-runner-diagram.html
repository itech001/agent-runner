Execution Pipeline
01

Load Agent Folder

Reads AGENTS.md, agent-runner.json, .env, and skills directory

02

Generate Plan

LLM creates step-by-step execution plan. Preview with --plan-only

03

Agent Loop

LLM call → tool execution → repeat until task_done or max iterations

04

Write Output

run.json, report, transcript, and trace log to output directory

Configuration

LLM_PROVIDER required

anthropic or openai

LLM_MODEL required

Model name e.g. claude-sonnet-4-20250514

LLM_BASE_URL optional

Override base URL for OpenAI-compatible APIs

ANTHROPIC_API_KEY if provider=anthropic

Anthropic API key

OPENAI_API_KEY if provider=openai

OpenAI API key

agent-runner.json config file

MCP servers, timeouts, permissions, agent behavior
