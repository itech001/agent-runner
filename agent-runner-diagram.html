Configuration

LLM_PROVIDER required

anthropic or openai

LLM_MODEL required

Model name e.g. claude-sonnet-4-20250514

LLM_BASE_URL optional

Override for OpenAI-compatible APIs

ANTHROPIC_API_KEY conditional

Required when provider is anthropic

OPENAI_API_KEY conditional

Required when provider is openai

agent-runner.json config

MCP servers, timeouts, permissions, agent behavior
